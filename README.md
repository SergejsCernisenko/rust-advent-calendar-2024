<h3>How to Use</h3>
<ol>
<li>Create a new task file: Simply create a new file in src/bin/ with any name (e.g., day02.rs, day03.rs, etc.)</li>
<li>Write your solution: Each file is a standalone program with its own main() function</li>
<li>Run a specific task:
cargo run --bin day01
Replace day01 with your filename (without the .rs extension)</li>
</ol>
